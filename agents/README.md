# Agents

This directory contains reference agents (ToyAgent, RateLimitAgent, ChainAgent, OpsTriageAgent)
that demonstrate how to interact with the benchmark tasks.

- Read `docs/agents/agents.md` for a catalog covering each agent’s goals, strategies,
  and when to use them.
- Bring your own implementation by following the contract in
  `docs/agents/agent_interface.md`.

> Terminal Bench, but for agents that actually have to do things.
